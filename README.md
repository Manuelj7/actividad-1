# Actividad 1 – Fundamentos de Python

Este repositorio contiene 5 programas en Python que resuelven diferentes ejercicios básicos de programación.
Cada ejercicio se encuentra en un archivo independiente.

Para ejecutar los programas es necesario tener **Python instalado** en el computador.

---

# Cómo ejecutar los programas

1. Abrir la terminal o consola.
2. Ir a la carpeta donde están los archivos.
3. Ejecutar el programa con el comando:

python nombre_del_archivo.py

Ejemplo:

python ejercicio1_estudiantes.py

---

# Ejercicio 1 – Gestión de estudiantes

Archivo:

ejercicio1_estudiantes.py

## Qué hace el programa

Este programa permite ingresar información de uno o varios estudiantes.
Para cada estudiante se registran:

* Nombre
* Edad
* Estado (activo o inactivo)
* Materias
* Notas por materia

Luego el programa calcula:

* Promedio general
* Materia con mejor nota
* Materia con peor nota
* Si el estudiante aprueba (promedio mayor o igual a 3.0)

## Cómo usarlo

1. Ejecutar el archivo.
2. Ingresar la cantidad de estudiantes.
3. Ingresar los datos que el programa solicita.
4. El programa mostrará los resultados en pantalla.

# Ejercicio 2 – Cálculo de impuesto y descuento

Archivo:

ejercicio2_impuesto.py

## Qué hace el programa

El programa recibe un valor monetario y calcula:

* Impuesto del 19%
* Descuento del 10% si el valor es mayor o igual a 200000
* Total final a pagar

## Cómo usarlo

1. Ejecutar el programa.
2. Ingresar el valor base cuando el programa lo solicite.
3. El programa mostrará el impuesto, el descuento y el total final.

# Ejercicio 3 – Serie Fibonacci

Archivo:

ejercicio3_fibonacci.py

## Qué hace el programa

Este programa genera la serie de Fibonacci y guarda los valores en una lista.

Al final muestra:

* La lista completa generada
* Cantidad de términos
* Último valor de la serie

## Cómo usarlo

1. Ejecutar el programa.
2. Ingresar el número inicial.
3. Ingresar la cantidad de iteraciones.
4. El programa mostrará la serie generada.

Si se ingresa un número negativo, el programa mostrará un mensaje de error.

# Ejercicio 5 – Validación de datos

Archivo:

ejercicio5_validacion.py

## Qué hace el programa

El programa solicita los siguientes datos:

* Nombre
* Edad
* Correo electrónico

El programa valida que:

* El nombre tenga mínimo 3 caracteres.
* La edad esté entre 0 y 120.
* El correo contenga "@", ".com" o "edu.co".

Si algún dato es incorrecto, el programa lo vuelve a pedir.

Finalmente los datos se almacenan en una lista.

## Cómo usarlo

1. Ejecutar el programa.
2. Ingresar los datos solicitados.
3. Si algún dato es inválido, el programa pedirá ingresarlo nuevamente.
4. El usuario registrado se mostrará en pantalla.
